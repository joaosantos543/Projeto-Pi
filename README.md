- JDBC (Java Database Connectivity)

 Objetivo do Software
O sistema tem como objetivo gerenciar produtos em uma **casa de leilões**, permitindo o **cadastro, listagem e atualização de itens**, além de controlar seu **status** (à venda ou vendido).  
Ele visa agilizar o processo de administração dos produtos leiloados e fornecer maior organização no controle de estoque.

 Funcionalidades do Sistema
- Cadastro de novos produtos no banco de dados.  
- Listagem de todos os itens cadastrados, exibindo nome, valor e status.  
- Alteração do status do produto (ex.: "À Venda"  "Vendido").  
- Mensagens de confirmação para sucesso ou falha no cadastro.  
- Integração com banco de dados **uc11** (tabela `produtos`).
